## Shortly - A Site That Shortens URLs

This is a project that aims to make URLs shorter.

### Built With:

- Next.js,
- TypeScript,
- Tailwind CSS,
- daisyUI,
- Next.js API Routes,
- Axios.

### Used Methods/Hooks:

- useState,
- useEffect,
- map(),
- Conditional Rendering (&& and ?),
- Array destructuring,
- POST API route,
- POST method.

Check it out on [here](https://conference-ticket-gen-brown.vercel.app) !

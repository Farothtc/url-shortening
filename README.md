## Shortly - A Site That Shortens URL's

This is a project that aims to make URLs shorter.

### Built With:

- Next.js,
- TypeScript,
- Tailwind CSS,
- daisyUI.

### Used Methods/Hooks:

- useState,
- useEffect,
- map(),
- Conditional Rendering (&& and ?),
- Array destructuring,
- POST API route,
- POST method.

Check it out on [here](https://conference-ticket-gen-brown.vercel.app) !
